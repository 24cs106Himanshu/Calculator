# 🎉 Happy Birthday Mom Website 🎂

A beautiful, animated birthday website dedicated to celebrating your amazing mom! This website features interactive elements, photo spaces, animations, and heartfelt messages.

## 🌟 Features

- **Animated Background**: Floating hearts, sparkles, and confetti
- **Photo Gallery**: 9 dedicated spaces for different types of photos
- **Interactive Candles**: Click to blow out candles and make a wish
- **Message Cards**: Beautiful cards highlighting why mom is amazing
- **Music Controls**: Background music toggle (you can add your own audio file)
- **Photo Upload**: Click on any photo placeholder to upload images
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Celebration Effects**: Various animations and particle effects

## 🚀 How to Set Up

### Option 1: Simple Setup
1. Download all files (`mom-birthday.html`, `mom-birthday.css`, `mom-birthday.js`)
2. Put them in the same folder
3. Double-click `mom-birthday.html` to open in your browser
4. Click on photo placeholders to upload images

### Option 2: With Background Music
1. Find a birthday song MP3 file
2. Name it `happy-birthday.mp3`
3. Put it in the same folder as the other files
4. Uncomment the audio source line in the HTML file:
   ```html
   <source src="happy-birthday.mp3" type="audio/mpeg">
   ```

### Option 3: Web Server Setup
If you want to share online:
1. Upload all files to a web hosting service
2. Share the URL with family and friends
3. Everyone can view the birthday tribute online

## 🔧 Technical Notes

### Browser Compatibility
- Works in all modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers fully supported
- No plugins or special software required

### File Formats Supported
- **Photos**: JPG, PNG, GIF, WebP
- **Audio**: MP3, WAV, OGG (for background music)

### Performance Tips
- Optimize large photos before uploading (recommended max: 2MB per photo)
- Use JPG format for photographs, PNG for graphics with transparency
- Test on mobile devices to ensure smooth animations
